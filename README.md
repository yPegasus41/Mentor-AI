# 🎓 Mentor-AI: Mentor de Estudos com Inteligência Artificial

[![Licença: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Status do Projeto: Em Desenvolvimento](https://img.shields.io/badge/Status-Em%20Desenvolvimento-green)

O **Mentor-AI** é uma aplicação web desenvolvida para auxiliar estudantes na organização e esclarecimento de dúvidas através de uma interface inteligente. O projeto foca em unir uma experiência de usuário (UX) moderna com a funcionalidade de processamento de linguagem natural via API.

---

## 🎯 Objetivo do Projeto

Este projeto nasceu da necessidade de centralizar o suporte ao estudo em uma interface amigável. Foi desenvolvido como parte do meu portfólio para aplicar conceitos de **Sistemas de Informação** e **Desenvolvimento Full Stack**, integrando lógica de backend com uma interface responsiva.

---

## 🛠️ Tecnologias e Ferramentas

O projeto utiliza uma stack leve e eficiente:

* **Frontend:** HTML5 semântico, CSS3 (Flexbox/Grid) e JavaScript Vanilla.
* **Backend:** PHP (Processamento de requisições e segurança).
* **Consumo de API:** Integração para inteligência artificial.
* **Design:** UI focada em legibilidade e modo escuro/claro (opcional).

---

## 🚀 Funcionalidades Principais

* **Interface de Chat:** Layout intuitivo inspirado em ferramentas modernas de IA.
* **Responsividade:** Otimizado para navegação em smartphones e desktops.
* **Segurança no Backend:** Scripts PHP para tratar os dados antes do envio para a API.
* **Feedback Visual:** Indicadores de carregamento e estados de erro amigáveis.

---

## 📦 Como Instalar e Testar

1.  **Clone este repositório:**
    ```bash
    git clone [https://github.com/yPegasus41/Mentor-AI.git](https://github.com/yPegasus41/Mentor-AI.git)
    ```

2.  **Configure o Ambiente:**
    * Certifique-se de ter um servidor PHP ativo (XAMPP, Laragon ou similar).
    * Coloque a pasta na raiz do seu servidor (ex: `htdocs` ou `www`).

3.  **Configuração da API:**
    * (Opcional) Renomeie o arquivo de configuração (se houver) e insira sua chave de API.

4.  **Acesse no Navegador:**
    ```text
    http://localhost/Mentor-AI
    ```
